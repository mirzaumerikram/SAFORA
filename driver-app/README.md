# SAFORA Driver App

React Native mobile application for drivers.

## Features
- Driver registration with vehicle information
- Accept/reject ride requests
- Real-time GPS tracking
- Earnings dashboard
- Rating system
- Navigation integration

## Setup
```bash
# Install dependencies
npm install

# Run on Android
npm run android

# Run on iOS
npm run ios
```
