# SAFORA Passenger App

React Native mobile application for passengers.

## Features
- User registration and authentication
- Ride booking with real-time tracking
- Pink Pass enrollment and verification
- SOS emergency button
- Ride history and ratings
- Payment management

## Setup
```bash
# Install dependencies
npm install

# Run on Android
npm run android

# Run on iOS
npm run ios
```
